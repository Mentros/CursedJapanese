# CursedJapanese

Angular web app for learning Japanese with PrimeNG + PrimeFlex.

## MVP Features

- Kana practice (hiragana / katakana / mixed)
- Flashcards with spaced repetition scheduling
- Progress dashboard with reset option
- Local-only persistence in browser storage

## Tech Stack

- Angular (standalone components)
- PrimeNG
- PrimeFlex
- localStorage for saved progress

## Run Locally

```bash
npm install
npm start
```

Then open `http://localhost:4200`.

## Build

```bash
npm run build
```
